Que es handshake?

En el contexto de las redes Wi-Fi protegidas con WPA/WPA2, el handshake es el proceso de intercambio de mensajes entre un dispositivo cliente (como una computadora portátil, un teléfono inteligente u otro dispositivo conectado a la red) y el punto de acceso (router) de la red Wi-Fi.

Cuando un dispositivo cliente intenta conectarse a una red Wi-Fi protegida, el punto de acceso envía un desafío al dispositivo para verificar su identidad. El dispositivo cliente responde a este desafío con información de autenticación y, si la autenticación es exitosa, se establece la conexión a la red. Este intercambio de mensajes entre el dispositivo cliente y el punto de acceso se conoce como "handshake".

El handshake capturado contiene información importante que puede utilizarse para realizar ataques de cracking de contraseñas en la red Wi-Fi. Este handshake capturado es crucial para intentar descifrar la contraseña de la red mediante técnicas de fuerza bruta o de diccionario. Por lo tanto, el proceso de capturar y utilizar el handshake es fundamental en muchos ataques contra redes Wi-Fi protegidas.
