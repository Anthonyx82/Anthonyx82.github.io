¿Qué es el Handshake?

En el contexto de las redes Wi-Fi protegidas con WPA/WPA2, el handshake es el proceso de intercambio de mensajes entre un dispositivo cliente (como un ordenador portátil, un teléfono inteligente u otro dispositivo conectado a la red) y el punto de acceso (router) de dicha red.

Cuando un dispositivo intenta conectarse a una red Wi-Fi protegida, el punto de acceso le envía un desafío para verificar su identidad. El dispositivo responde con información de autenticación y, si esta es válida, se establece la conexión. Este intercambio se denomina "handshake".

Un handshake capturado puede contener información útil para intentar descifrar la contraseña de la red, mediante técnicas como fuerza bruta o ataques de diccionario. Por ello, es una parte crítica en los ataques dirigidos a redes Wi-Fi protegidas.
